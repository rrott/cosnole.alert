cosnole.alert()

Extends browser's console and window objects to:

  • show toast messages on all console.log() calls
  • attach alert() function to the console object
  • stop js runtime and call alert() on all or pre-defined console.log() calls
  • attach custom methods to the console and window object
  • run pre- and after-hooks methods around console.log() call
  • etc.. (See documentation)
